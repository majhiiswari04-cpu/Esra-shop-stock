# My Shop

A simple shopping website connected with Google Sheets.

## Features

- Live Product List
- Product Name
- Price
- Search
- Mobile Friendly
- Google Sheets Auto Update

## Technologies

- HTML
- CSS
- JavaScript
- Google Apps Script
- GitHub Pages

## Setup

1. Create a Google Sheet.
2. Deploy Google Apps Script as a Web App.
3. Copy the Web App URL.
4. Paste the URL in `script.js`.
5. Upload `index.html`, `style.css`, and `script.js` to GitHub.
6. Enable GitHub Pages.
